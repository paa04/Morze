## Введение

Документ описывает жизненный цикл проекта по методологии p3express для desktop-мессенджера с клиентской и серверной частями. Проект включает личные чаты через P2P, групповые чаты через relay server, signaling для WebRTC и локальное хранение истории сообщений.

## Правила обновления документа

- Формат версии документа: `MAJOR.MINOR.PATCH`.
- Увеличение `MAJOR` выполняется при изменении структуры этапов или логики жизненного цикла проекта.
- Увеличение `MINOR` выполняется при изменении содержания отдельных действий или разделов.
- Увеличение `PATCH` выполняется при редакционных и оформительских правках без изменения смысла.
- При любом изменении документа поле **Дата последнего обновления** должно быть актуализировано в формате `DD-MM-YYYY`.
